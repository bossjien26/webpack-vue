# webpack-vue

use webpack vue create store , connect rest api

## Back-End project example

[back-end project](https://github.com/bossjien26/Rest-Api)

## environment

> install npm , vue-cli

## install package

```sh
npm install
```

## start up

### development

- setting .env file

```sh
npm run dev
```

### build docker

```sh
#currently env is dev
docker-compose up -d cart_web
```
